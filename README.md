# Scademy website

> Đây là dự án thiết kế trang ELearning, sử dụng server actions của NextJS 14.

### Công nghệ sử dụng:

- NextJS 14, Typescript, Tailwind CSS, Shadcn UI, Clerk, MongoDB, Mongoose, Uploadthing.

### Lưu ý:

- Khi truy cập trang web lần đầu tiên, có thể mất một chút thời gian để tải dữ liệu.

### Chức năng:

- Trang chủ hiển thị tất cả các khóa học đã được tạo.

- Trang khóa học đã đăng ký và trang học các video.

- Trang quản lý khóa học: tìm kiếm, lọc, duyệt các khóa học, thêm và cập nhật các khóa học, thêm và cập nhật các chương với bài học.

- Trang quản lý đơn hàng: tìm kiếm, lọc, duyệt đơn hàng.

- Sử dụng Clerk để quản lý phân quyền, đăng nhập, đăng ký và thông tin người dùng.

- Sử dụng Uploadthing lưu các hình ảnh.
