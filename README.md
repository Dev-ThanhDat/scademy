# Scademy website

> This is an ELearning page design project, using server actions of NextJS 14.

### Technology used:

- NextJS 14, Typescript, Tailwind CSS, Shadcn UI, Clerk, MongoDB, Mongoose, Uploadthing.

### Function:

- The home page shows all the courses that have been created.

- Registered course page and learning videos page.

- Course management page: search, filter, browse courses, add and update courses, add and update chapters with lessons.

- Order management page: search, filter, browse orders.

- Use Clerk to manage authorization, login, registration and user information.

- Use Uploadthing to save the images.
